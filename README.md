######MVP的优点：

  - 模型与视图完全分离，我们可以修改视图而不影响模型

  - 可以更高效地使用模型，因为所有的交互都发生在一个地方——Presenter内部

  - 我们可以将一个Presenter用于多个视图，而不需要改变Presenter的逻辑。这个特性非常的有用，因为视图的变化总是比模型的变化频繁。

  - 如果我们把逻辑放在Presenter中，那么我们就可以脱离用户接口来测试这些逻辑（单元测试）
